# Front End/Client untuk POS Jualin
Jualin adalah POS open source. 

Guide:
#1 Clone repository
#2 Install menggunakan npm/pnpm
    - npm i
    - pnpm i
#3 Buat file .env di root project(level README.md, index.html, package.json, dll). Lalu isi sesuai contoh:
    VITE_BACKEND_API=http://localhost:3000
